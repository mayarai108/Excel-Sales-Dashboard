#  Project Overview

This project demonstrates an end-to-end Sales Data Analytics solution developed in Microsoft Excel. The dashboard transforms raw sales data into meaningful business insights using Power Query, Power Pivot, DAX Measures, Pivot Tables, Pivot Charts, VBA Macros, and Interactive Dashboards.

The primary objective of this project is to automate sales reporting, monitor key performance indicators (KPIs), and support data-driven business decisions.

-------------------------

#  Business Problem

Businesses often store sales data in raw tabular format, making it difficult to analyze trends and monitor performance.

This project solves that problem by converting raw data into an interactive dashboard that enables users to:

- Monitor overall sales performance
- Track monthly sales trends
- Identify top-performing products
- Compare regional sales
- Analyze customer purchases
- Filter data dynamically

# 📂 Project Workflow

## 1️. Data Import

- Imported raw sales dataset into Microsoft Excel.

## 2️. Data Cleaning (Power Query)

Performed data transformation using Power Query:

- Removed duplicate records
- Changed data types
- Renamed columns
- Cleaned inconsistent values
- Prepared data for analysis

## 3️.Data Modeling
- Loaded cleaned data into Excel Data Model
- Created relationships between tables
- Built a separate Calendar Table for date intelligence

## 4️. KPI Creation (Power Pivot)

Created DAX Measures for:

* Total Sales
* Total Revenue
* Total Discount
* Total Orders
* Top Product
* Top City

## 5️. Data Analysis

Created multiple Pivot Tables to analyze:

- Monthly Sales
- Regional Sales
- Product-wise Sales
- Category-wise Sales
- Customer-wise Sales
- City-wise Sales

## 6️. Dashboard Development

Designed an interactive dashboard including:

- KPI Cards
- Line Chart
- Column Chart
- Donut Chart
- Pivot Charts
- Timeline Filter
- Slicers
- Interactive Filters

## 7️. Automation

Implemented VBA Macros to automate dashboard navigation and improve user experience.

## 8️.Documentation

Prepared a complete project report in Microsoft Word explaining:

- Business Objective
- Methodology
- Dashboard Design
- Key Insights
- Conclusion
